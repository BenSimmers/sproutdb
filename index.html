SproutDB

A tiny, dependency-free TypeScript key-value store with pluggable persistence and advanced querying

TypeScript First Zero Dependencies REST API Advanced Queries

Install via npm

npm install sproutdb

Try instantly with npx

As a Library

import { create, table } from 'sproutdb'; const db = create({ users: table<{ id: number; name: string; email: string }>(), posts: table<{ id: number; title: string; content: string }>(), }); // Insert data db.users.insert({ id: 1, name: 'Alice', email: 'alice@example.com' }); // Find with advanced queries const adults = db.users.find({ where: { age: { $gte: 18 } }, sort: { name: 'asc' }, limit: 10 }); // Complex conditions const result = db.users.find({ where: { $or: [ { name: { $regex: /^A/ } }, { age: { $lt: 30 } } ] } });

üìö API Reference

Database Operations

Method Description
create(schema) Creates a new database instance with table schema
table<T>() Creates a new table for records of type T

Table Methods

Method Parameters Description
insert(record) T Inserts a new record into the table
find(query?) QueryOptions<T> Finds records with advanced filtering, sorting, pagination
all() Returns all records in the table
update(query, update) QueryOptions<T>, Partial<T> Updates records matching the query
delete(query) QueryOptions<T> Deletes records matching the query
load(records) T[] Loads multiple records into the table

Query Operators

Operator Description Example
$gt Greater than { age: { $gt: 18 } }
$gte Greater than or equal { age: { $gte: 21 } }
$lt Less than { age: { $lt: 65 } }
$lte Less than or equal { age: { $lte: 65 } }
$ne Not equal { status: { $ne: 'inactive' } }
Operator Description Example
$or Logical OR { $or: [{ name: 'Alice' }, { age: { $lt: 30 } }] }
Operator Description Example
$in In array { status: { $in: ['active', 'pending'] } }
$nin Not in array { role: { $nin: ['admin', 'moderator'] } }
$regex Regular expression { email: { $regex: /@example\.com$/ } }
Option Type Description Example
where Partial<T> | WhereClause<T> Query conditions { age: { $gte: 18 } }
sort SortClause<T> Sort specification { name: 'asc', age: 'desc' }
limit number Maximum results 10
offset number Skip results 20
